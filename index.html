Otherapp/ropbin payloads


Some exploits (not ninjhax, ironhax or tubehax) might allow you to run the homebrew launcher and require you to get an "otherapp payload" (or a "ropbin payload") to do so. This payload is a binary file specific to your firmware version. You can download your own payload by selecting your firmware version below (only select NEW if you are running this on a New 3DS (XL); regular 3DS, 3DS XL and 2DS owners should select OLD) :
